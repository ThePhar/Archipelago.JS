Class ArchipelagoClient

The client that connects to an Archipelago server and facilitates communication, listens for events, and manages data.

constructor

  • new ArchipelagoClient(socketAddress: string, secure?: boolean): ArchipelagoClient

  • Creates a new client that is programmed to connect to a specific Archipelago server address.

    Parameters

    • socketAddress: string

      The socket address to connect to. Examples: 127.0.0.1:50000 or archipelago.gg:38281.

    • secure: boolean = false

      Whether to attempt to connect with wss:// instead of ws://.

    Returns ArchipelagoClient

connect

  • connect(credentials: SlotCredentials): Promise<void>

  • Connects to the given address with given connection information.

    Resolves

    On successful connection and authentication to the room.

    Rejects

    If web socket connection failed to establish connection or server refused connection, promise will return a string[] of error messages.

    Parameters

    • credentials: SlotCredentials

      An object with all the credential information to connect to the slot.

    Returns Promise<void>
