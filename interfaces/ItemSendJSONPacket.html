Interface ItemSendJSONPacket

Sent to clients purely to display a message to the player. This packet differs from a PrintPacket in that the data being sent with this packet allows for more configurable or specific messaging.

This is a more specific version of PrintJSONPacket that includes required fields for PrintJSONType.ITEM_SEND type packets.
